Для запуска кода потребуется:
1) lazarus 1.4: lazarus-1.4.0-fpc-2.6.4-win32.exe
2) fpc 1.6.4: lazarus-1.4.0-fpc-2.6.4-win32.exe
3) Пакет библиотек под процессов arm: lazarus-1.4.0-fpc-2.6.4-cross-arm-wince-win32.exe

Для корректного запуска плеера в каталоге с программой должны размещаться следующие билиотеки:
1) aygshell.dll размером 22kb
2) bass.dll
3) bass_aac.dll
4) bass_fx.dll
5) bassalac.dll
6) bassflac.dll
7) tags.dll

А так же не забудьте закинуть папку "Singleplayer" из архива с бинарником в папку с компилируемой программой.
